# trigonometry
Визуализация единичной окружности и тригонометрических функций.
## Что реализовано

* Единичная окружность (с использованием собственных классов `Point` и `Circle`)
* Графики функций `sin(α)` и `cos(α)` на интервале от `-2π` до `2π`
* Визуальная связь между кругом и графиками

## Детали реализации

1. Окружность строится параметрически:
   * `X = R·cos(α)`
   * `Y = R·sin(α)`
2. Графики функций:
   * `cos(α)` — проекция на ось `X`
   * `sin(α)` — проекция на ось `Y`

## Ключевая идея

Синус и косинус — это не абстрактные функции,  
а просто координаты точки, бегущей по кругу.
